# 项目背景

:smile:  项目来源于 **开课吧** java基础课程，原课程需要云服务器及POLARDB数据库搭建，及微信公众号部署，是一个可以写进简历的项目。这里根据相关代码修改简化，用本地数据库搭建，不再增加付费环节。
